## Community Maintenance Platform System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/31b1949f40c941c8a97276e7a86f3c81/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): residents, maintenance workers, and administrators.

**Regular User Features:**
1. Login
2. Register
3. Homepage
4. Community Announcements
5. Maintenance Workers
6. Maintenance Orders, Order Taking, Communication
7. News Information
8. Message Feedback
9. Personal Center
10. Post Maintenance Request
11. My Order Taking Information
12. My Order Information
13. Online Communication
14. My Report Information

**Maintenance Worker Features:**
1. Login
2. Register
3. Homepage
4. Community Announcements
5. Maintenance Workers
6. Maintenance Orders, Order Taking, Communication
7. News Information
8. Message Feedback
9. Personal Center
10. My Order Taking Information
11. My Order Information
12. Online Communication

**Administrator Features:**
1. Login
2. Personal Center
3. Resident Management
4. Community Announcement Management
5. Maintenance Worker Management
6. Maintenance Order Management
7. Order Taking Information Management
8. Order Information Management
9. Online Communication Management
10. Report Information Management
11. Message Board Management
12. News Information Management
13. Carousel Management
